# AI Image Editor Pro

A professional AI-powered image editor built with Flask, OpenCV, and rembg.

## Features

- Background Removal
- Background Replacement
- Cartoon Filter
- Pencil Sketch
- HDR Effect
- Brightness & Contrast
- Resize Images
- PNG/JPG/WEBP Conversion

## Tech Stack

- Python
- Flask
- OpenCV
- rembg
- Bootstrap 5

## Installation

```bash
pip install -r requirements.txt
python main.py
